# Demo Cases

This folder contains two ready-to-run demo CT studies for `OsteoVigil`.

## Included Cases

### 1. `normal_real_talocrural`

- Type: real public DICOM CT
- Anatomy: distal tibia, distal fibula, and ankle region
- Contents: `325` DICOM slices
- Approximate size: `100 MB`
- Source:
  - Article: https://www.nature.com/articles/s41598-021-86567-7
  - PMC mirror: https://pmc.ncbi.nlm.nih.gov/articles/PMC8016855/
  - Zenodo record: https://doi.org/10.5281/zenodo.4274217
- Provenance note:
  - The included DICOM files were extracted from the `WeightbearingCT_Sample/` folder inside `Lenz-Lab/TalocruralShapeModeling-v1.0.zip`.
  - The article's data-availability statement says the dataset can be used under `CC0`.
- Important limitation:
  - This is a normal distal tibia/fibula/ankle case, not a full tibial shaft study.

### 2. `abnormal_synthetic_cpt`

- Type: synthetic DICOM CT generated by this repository
- Anatomy: lower-leg style tibia/fibula field with a simulated pseudarthrosis-like defect
- Contents: `180` DICOM slices plus `afo_proxy.stl`
- Approximate size: `7.7 MB`
- Source:
  - Generated locally by `src/cpt_predictor/io/sample_data.py`
- Important limitation:
  - This is not a real patient scan. It is intended only as a reproducible abnormal demo case.

## Quick Run Commands

### Real normal case

```bash
python main.py \
  --dicom-dir data/demo/normal_real_talocrural \
  --output-dir outputs/demo_normal_real
```

### Synthetic abnormal CPT case

```bash
python main.py \
  --dicom-dir data/demo/abnormal_synthetic_cpt/dicom \
  --brace-stl data/demo/abnormal_synthetic_cpt/afo_proxy.stl \
  --output-dir outputs/demo_abnormal_synthetic
```

### With agent orchestration

```bash
python main.py \
  --dicom-dir data/demo/normal_real_talocrural \
  --use-agents \
  --output-dir outputs/demo_normal_real_agents
```

## Validation Performed

Both demo folders were loaded successfully using `src/cpt_predictor/io/dicom_loader.py`.

- `normal_real_talocrural`: shape `(325, 400, 400)`, spacing `(0.4, 0.4, 0.4)`
- `abnormal_synthetic_cpt`: shape `(180, 144, 144)`, spacing `(1.1, 0.8, 0.8)`
